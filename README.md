# Remaps

|                                 |             |
|---------------------------------|-------------|
| Leader Key                      | "Space"     | 
| Open File Explorer (nvim-tree)  | <leader>b   |
| Search File (telescope)         | <leader>p   |
| Search Git File (telescope)     | <C-p>       |
| Search within files (telescope) | <leader>f   |
| 

|                            | Normal Mode | Visual Mode | Insert Mode  |
|----------------------------|-------------|-------------|--------------|
| Move Line(s) Up            | K           | K           |              |
| Move Line(s) Down          | J           | J           |              |
